# 이터널리턴 실험체 점수 계산기

이터널리턴에서 특정 실험체(캐릭터)의 숙련도를 점수로 계산하는 웹 계산기입니다.

## 기능

- 승률, TOP3, RP, 평균순위, 평균킬 5가지 지표로 점수 계산
- 평균 vs 플레이어 비교 그래프
- 가중치 조정 가능 (기본 각 20%)
- SS~F 등급 자동 부여

## 사용 방법

1. **평균 유저 데이터** 입력 (해당 티어의 실험체 평균 통계)
2. **플레이어 데이터** 입력 (분석하려는 유저의 전적)
3. **가중치 설정** (선택사항, 합계 100%)
4. **점수 계산** 버튼 클릭

## 실행

`calculator.html` 파일을 브라우저로 열면 바로 실행됩니다.

## 점수 해석

- **100점**: 평균과 동일한 수준
- **150점 이상**: SS등급 (압도적)
- **90~149점**: S등급 (매우 우수)
- **80~89점**: A등급
- **70~79점**: B등급
- **60~69점**: C등급
- **50~59점**: D등급
- **50점 미만**: F등급

## 기술

HTML, JavaScript, Tailwind CSS, Chart.js

## 주의사항

- 정확한 데이터 입력 필요
- 티어가 비슷한 평균 데이터와 비교해야 정확함
- 최소 20게임 이상 데이터 권장
